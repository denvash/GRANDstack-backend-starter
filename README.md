# GraphQL API

## Deployments

- [Neo4j](https://10-0-1-104-35408.neo4jsandbox.com/browser/)
- [GraphQL Server](https://core.dennisvash.now.sh/graphql)

## Quick Start

1. Install Dependencies

    ```sh
    yarn install
    ```

2. Development

    ```sh
    yarn develop
    # Run for seeding the Neo4j DB
    yarn seed
    ```

3. Deployment

    ```sh
    npm i -g now
    now login
    now
    ```
